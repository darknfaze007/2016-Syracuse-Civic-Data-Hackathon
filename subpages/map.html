Below is a map of 7,392 data points where potholes have been filled in the Syracuse area by a machine called a durapatcher. I used the pothole data spread sheet to extract the GPS coordinants and apply them to the Google Map API. This data was hosted on a Google Fusion table, a free data table hosting application provided by Google which can be seen here.

Source: https://cityofsyracuse.github.io/RoadsChallenge/data/potholes.csv
